# HaoChiUtils.py

"浩池懒做"的NLP工具库

## DataPreprocess

数据预处理类

1. text_process() :将文本清洗后存入指定路径文件中
2. text_clean()： 将一个字符串文本清洗后，返回清洗后的结果

## DataAnalyzer

数据分析类

1. split_dataSet(): 划分数据集为测试集、验证集、训练集
2. draw_process(): 绘制训练过程的曲线
3. calculate_label_proportions(): 计算标签占比，输入为预测结果的列表,输出为 标签:占比
